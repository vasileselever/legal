## Request Flow Example

### Creating a Case:

```
1. Client (Browser/App)
   POST /api/v1/cases
   Authorization: Bearer eyJhbGciOiJIUzI1NiIs...
   Body: { title: "Contract Dispute", clientId: "...", ... }
   ?
   ?
2. API Gateway
   - HTTPS termination
   - Rate limiting
   - Load balancing
   ?
   ?
3. ASP.NET Core Middleware Pipeline
   ??? JWT Authentication (validates token)
   ??? Authorization (checks permissions)
   ??? CORS (validates origin)
   ??? Request logging
   ??? Error handling
   ?
   ?
4. CasesController
   - Validates request
   - Extracts user claims (UserId, FirmId)
   ?
   ?
5. CaseService (Business Logic)
   - Validates client belongs to firm
   - Validates lawyer belongs to firm
   - Generates case number
   - Creates Case entity
   ?
   ?
6. ApplicationDbContext (EF Core)
   - Adds Case to DbSet
   - Adds Activity log
   - SaveChangesAsync()
   ?
   ?
7. SQL Server
   - BEGIN TRANSACTION
   - INSERT INTO Cases...
   - INSERT INTO Activities...
   - COMMIT
   ?
   ?
8. Response Flow (reverse)
   SQL Server ? EF Core ? Service ? Controller ? Middleware ? API Gateway ? Client
   
   200 OK
   Body: { id: "...", caseNumber: "2024-00001", ... }
```

## Technology Stack Summary

| Layer | Technology | Purpose |
|-------|-----------|---------|
| **Frontend** | Blazor WebAssembly / React | Web application UI |
| **Mobile** | .NET MAUI | iOS/Android apps |
| **API** | ASP.NET Core 8 Web API | RESTful API endpoints |
| **Authentication** | JWT Bearer + Identity | User authentication |
| **ORM** | Entity Framework Core 8 | Database access |
| **Database** | SQL Server 2019+ | Relational data storage |
| **File Storage** | Azure Blob Storage | Document storage |
| **Caching** | Redis | Session & query caching |
| **Logging** | Serilog | Structured logging |
| **Documentation** | Swagger/OpenAPI | API documentation |
| **Validation** | FluentValidation | Input validation |
| **Mapping** | AutoMapper | DTO mapping |
| **Cloud** | Azure | Cloud hosting |
| **CI/CD** | Azure DevOps / GitHub Actions | Automated deployment |
